## How to Read the Charts
- **Bar charts** show contact hours by organization for a given year
- **Non-LEAD** charts exclude LEAD organizations for a focused comparison
- **YoY chart** shows trends across modalities over time

## Code
The analysis is written in R. To run it:

1. Open `SF Org Identifiers Merge.R` first — this prepares the organization identifiers
2. Then run `Contact Hours Analysis.R` to generate the charts
